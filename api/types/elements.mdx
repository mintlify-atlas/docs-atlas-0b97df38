## ExcalidrawElement

The base union type for all Excalidraw elements. All elements are JSON serializable and contain no computed data.

```typescript
type ExcalidrawElement =
  | ExcalidrawGenericElement
  | ExcalidrawTextElement
  | ExcalidrawLinearElement
  | ExcalidrawArrowElement
  | ExcalidrawFreeDrawElement
  | ExcalidrawImageElement
  | ExcalidrawFrameElement
  | ExcalidrawMagicFrameElement
  | ExcalidrawIframeElement
  | ExcalidrawEmbeddableElement
```

## Base Element Properties

All element types extend from `_ExcalidrawElementBase` with these common properties:

<ResponseField name="id" type="string" required>
  Unique identifier for the element
</ResponseField>

<ResponseField name="x" type="number" required>
  X coordinate of the element
</ResponseField>

<ResponseField name="y" type="number" required>
  Y coordinate of the element
</ResponseField>

<ResponseField name="strokeColor" type="string" required>
  Color of the element's stroke
</ResponseField>

<ResponseField name="backgroundColor" type="string" required>
  Background color of the element
</ResponseField>

<ResponseField name="fillStyle" type="FillStyle" required>
  Fill style: `"hachure"` | `"cross-hatch"` | `"solid"` | `"zigzag"`
</ResponseField>

<ResponseField name="strokeWidth" type="number" required>
  Width of the element's stroke
</ResponseField>

<ResponseField name="strokeStyle" type="StrokeStyle" required>
  Stroke style: `"solid"` | `"dashed"` | `"dotted"`
</ResponseField>

<ResponseField name="roundness" type="{ type: RoundnessType; value?: number } | null" required>
  Roundness configuration for the element
</ResponseField>

<ResponseField name="roughness" type="number" required>
  Roughness level for the hand-drawn effect
</ResponseField>

<ResponseField name="opacity" type="number" required>
  Opacity of the element (0-100)
</ResponseField>

<ResponseField name="width" type="number" required>
  Width of the element
</ResponseField>

<ResponseField name="height" type="number" required>
  Height of the element
</ResponseField>

<ResponseField name="angle" type="Radians" required>
  Rotation angle of the element in radians
</ResponseField>

<ResponseField name="seed" type="number" required>
  Random integer used to seed shape generation so that the roughjs shape doesn't differ across renders
</ResponseField>

<ResponseField name="version" type="number" required>
  Integer that is sequentially incremented on each change. Used to reconcile elements during collaboration or when saving to server
</ResponseField>

<ResponseField name="versionNonce" type="number" required>
  Random integer that is regenerated on each change. Used for deterministic reconciliation of updates during collaboration, in case the versions are identical
</ResponseField>

<ResponseField name="index" type="FractionalIndex | null" required>
  String in a fractional form defined by fractional-indexing. Used for ordering in multiplayer scenarios. Could be null for new elements which were not yet assigned to the scene
</ResponseField>

<ResponseField name="isDeleted" type="boolean" required>
  Whether the element has been deleted
</ResponseField>

<ResponseField name="groupIds" type="readonly GroupId[]" required>
  List of groups the element belongs to, ordered from deepest to shallowest
</ResponseField>

<ResponseField name="frameId" type="string | null" required>
  ID of the frame this element belongs to, if any
</ResponseField>

<ResponseField name="boundElements" type="readonly BoundElement[] | null" required>
  Other elements that are bound to this element
</ResponseField>

<ResponseField name="updated" type="number" required>
  Epoch (ms) timestamp of last element update
</ResponseField>

<ResponseField name="link" type="string | null" required>
  External link associated with this element
</ResponseField>

<ResponseField name="locked" type="boolean" required>
  Whether the element is locked from editing
</ResponseField>

<ResponseField name="customData" type="Record<string, any>">
  Custom metadata for the element
</ResponseField>

## Text Element

### ExcalidrawTextElement

Text element with rich formatting options.

<ResponseField name="type" type="'text'" required>
  Element type identifier
</ResponseField>

<ResponseField name="fontSize" type="number" required>
  Font size in pixels
</ResponseField>

<ResponseField name="fontFamily" type="FontFamilyValues" required>
  Font family identifier
</ResponseField>

<ResponseField name="text" type="string" required>
  The text content
</ResponseField>

<ResponseField name="textAlign" type="TextAlign" required>
  Text alignment: left, center, or right
</ResponseField>

<ResponseField name="verticalAlign" type="VerticalAlign" required>
  Vertical alignment within container
</ResponseField>

<ResponseField name="containerId" type="string | null" required>
  ID of the container element if text is bound to a shape
</ResponseField>

<ResponseField name="originalText" type="string" required>
  Original text before any wrapping or formatting
</ResponseField>

<ResponseField name="autoResize" type="boolean" required default="true">
  If `true` the width will fit the text. If `false`, the text will wrap to fit the width
</ResponseField>

<ResponseField name="lineHeight" type="number & { _brand: 'unitlessLineHeight' }" required>
  Unitless line height (aligned to W3C). To get line height in px, multiply with font size
</ResponseField>

<Expandable title="Inherits all base element properties">
  See [Base Element Properties](#base-element-properties) above.
</Expandable>

## Supporting Types

### BoundElement

Represents an element bound to another element.

<ResponseField name="id" type="string" required>
  ID of the bound linear element
</ResponseField>

<ResponseField name="type" type="'arrow' | 'text'" required>
  Type of the bound element
</ResponseField>

### FixedPointBinding

Binding information for arrow endpoints.

<ResponseField name="elementId" type="string" required>
  ID of the element being bound to
</ResponseField>

<ResponseField name="fixedPoint" type="[number, number]" required>
  Fixed point as [horizontal ratio, vertical ratio] in range 0.0-1.0
</ResponseField>

<ResponseField name="mode" type="BindMode" required>
  Binding mode: `"inside"` | `"orbit"` | `"skip"`. Determines whether the arrow remains outside the shape or goes inside
</ResponseField>

### Arrowhead

Arrowhead style options.

```typescript
type Arrowhead =
  | "arrow"
  | "bar"
  | "dot" // legacy
  | "circle"
  | "circle_outline"
  | "triangle"
  | "triangle_outline"
  | "diamond"
  | "diamond_outline"
  | "crowfoot_one"
  | "crowfoot_many"
  | "crowfoot_one_or_many"
```

### NonDeletedExcalidrawElement

Type helper for elements that are not deleted.

```typescript
type NonDeletedExcalidrawElement = NonDeleted<ExcalidrawElement>
```

### OrderedExcalidrawElement

Type helper for elements with a non-null index.

```typescript
type OrderedExcalidrawElement = Ordered<ExcalidrawElement>
```
